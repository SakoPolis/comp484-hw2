*Structural tags*
The aside tag is used for the W3C warning.
The article tag is used for the article on the page.
The nav tag is used for the links at the top to navigate to other parts of the site.
The section tag is used for dividing the document into semantic sections.

*Semantic tags*
The header is for the banner. The footer is for content information. The main element is for the body of the page.
